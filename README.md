Worth The Line? 🍸
A community-powered, real-time wait time app for NYC bars, lounges, and restaurants. Search any spot to see how long the line is right now — or report the wait yourself and help someone else plan their night.
Features

Find a Spot — search by venue name or filter by NYC neighborhood
Live venue cards — avg wait time, worth-it percentage, and report count at a glance
Tap for details — full breakdown of recent reports per venue
Report a Wait — submit your current wait in seconds
No login required. No tracking. Just real people, real lines.

Tech
Pure HTML, CSS, and JavaScript. No frameworks, no backend. Data stored via localStorage so it works entirely in the browser.
Run It
Download index.html and open it in any browser. That's it.
